# Day 6 - Vue 3 工作台

## 分工

- 前端骨架：Vue 3 + Vite + TS
- 交互区：会话、流程、对话、草稿、导出
- 视觉风格：暗色玻璃感教师工作台

## 目标

做一个真正可用的前端工作台，而不是展示页。

## 开发任务

- 建立 Vue 3 + Vite + TS 前端。
- 支持创建会话、切换流程、切换历史会话。
- 支持 SSE 流式聊天、草稿编辑、阶段查看、回滚与导出。
- 保持暗色玻璃感的教师工作台风格。

## 产物

- `frontend/`

## 验收标准

- 浏览器打开后第一屏就是工作台。
- 可以完整跑通一轮创建、对话、保存草稿、导出。
- 操作区和草稿区不会互相遮挡。
