# Day 2 - 流程与会话

## 分工

- 流程层：7 阶段科学探究流模板、阶段角色与说明
- 会话层：创建会话、查看详情、切换流程、会话列表
- 状态层：切换流程时重置阶段输出与专家上下文

## 目标

把 7 阶段科学探究流做成可切换、可追踪的会话状态机。

## 开发任务

- 定义流程模板、阶段列表和阶段专家角色。
- 支持创建会话、查看会话详情、切换流程。
- 切换流程时重置阶段输出和专家会话。
- 暴露会话列表，供前端切换历史记录。

## 产物

- `app/workflow/flows.py`
- `app/api/sessions.py`

## 验收标准

- 可以创建一个新会话并看到当前阶段。
- 可以在不同流程之间切换，且会话状态正确重置。
- 可以列出最近会话，按更新时间排序。

- 已完成并验证：流程列表、会话创建、会话列表、流程切换。
