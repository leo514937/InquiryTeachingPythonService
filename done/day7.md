# Day 7 - 联调与交付

## 分工

- 端到端联调：后端、前端、SSE、导出
- 交付收口：启动方式、环境变量、常见问题、验收记录

## 目标

把后端、前端、bootstrap 和文档串成一个可交付版本。

## 开发任务

- 做端到端联调，覆盖创建会话、切换流程、SSE 对话、保存草稿、回滚、导出。
- 检查无外部依赖时的 Mock 降级链路。
- 整理启动方式、环境变量和常见问题。
- 验证打包与本地启动流程。

## 产物

- `README.md`
- `bootstrap.py`
- 端到端验收记录

## 验收标准

- 干净环境下可以初始化数据库并启动服务。
- 前端能连接到后端并完成完整工作流。
- 导出的 Markdown 教案格式稳定，内容按阶段组织。
